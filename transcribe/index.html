ABOUT THIS PROJECT
What is this?

This is a fully client-side audio transcription system built with Transformers.js, powered by Whisper Large V3 Turbo running directly in your browser using WebGPU. Your audio files are never uploaded to any server - everything is processed locally on your device for maximum privacy.
